What is a media query?

A media query is an at-rule, but what is an at-rule?

At-rules are CSS statements, not related to styling elements, that tell CSS what to do.

You may be familiar with @import, @font-face and @keyframes, but there are quite a few more that you've probably never encountered before.

@import "./components/button.css";

font-face {
  font-family: "Open Sans";
  src: url("/fonts/OpenSans-Regular-webfont.woff2") format("woff2"),
       url("/fonts/OpenSans-Regular-webfont.woff") format("woff");
}
